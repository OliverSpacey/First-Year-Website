Game

How to play:

First, if you alter the window size make sure to refresh the page to ensure the game runs smoothly! Second, press the 'Start game!' button to run the game, where you will be shown 3 images and asked to select one. In order to restart the game, simply press the 'Start game!' button and enter your name again! The aim of the game is to get as many correct as you can, good luck!
